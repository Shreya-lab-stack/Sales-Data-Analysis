 # Sales Data Analysis using Python

## Objective
To analyze sales data and identify trends, top-performing products, and regional performance using Python.

## Dataset
Superstore Sales Dataset (9,800 records)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Year-wise and month-wise sales trends
- Category and region-wise performance
- Top 10 products by sales

## Key Insights
- Total sales amount to approximately **$2.26 million**
- Sales peak during **November and December**, indicating strong seasonality
- **Technology** category generates the highest revenue
- **West** region contributes the largest share of total sales
- Top 10 products account for a significant portion of overall revenue

## Conclusion
This analysis demonstrates how data-driven insights can support business decision-making.
